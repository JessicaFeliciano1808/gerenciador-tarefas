# gerenciador-tarefas
Sistema simples para gerenciamento de tarefas com foco em organização e controle de atividades
---

## 🛠️ Observação Técnica

Este projeto foi estruturado inicialmente com foco em lógica de negócio, organização e controle de tarefas.

A proposta pode evoluir para uma aplicação completa utilizando:
- Angular (front-end)
- ASP.NET Core (back-end)
- SQL Server (banco de dados)

---

## 💡 Visão do Projeto

A ideia central é demonstrar como a organização de tarefas e processos pode ser estruturada de forma simples, com possibilidade de evolução para sistemas mais robustos e automatizados.
